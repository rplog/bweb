# bweb

A terminal-based personal portfolio website built with React and Cloudflare Pages.

## Features

- Interactive terminal interface with command history
- Spotlight search (Press 'L' to toggle)
- Full-screen graphical pages for Gallery, About, and Contact
- Real-time ping simulation via Cloudflare Workers
- CRT-style visual effects

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Cloudflare Pages Functions

## Setup

1. Install dependencies:
   npm install

2. Run development server (with backend functions):
   npm run pages:dev

3. Build for production:
   npm run build
